# 시나리오 표지 refs (ORB 등록)

데모·시연에 쓰는 **4권**의 표지 이미지입니다. 웹/제스처 데모에서 카메라로 비추는 책과 **동일 표지**여야 `identify_book` ORB 매칭이 됩니다.

| refs 파일 | 제목 | 저자 | 시나리오 키 |
|-----------|------|------|-------------|
| `어른이된다는것.jpg` | 어른이 된다는 것 | 김창진 | `book1` |
| `오직두사람.jpg` | 오직 두 사람 | 김영하 | `book2` |
| `단한사람.jpeg` | 단 한 사람 | 최진영 | `serendipity` |
| `너무나많은여름이.jpg` | 너무나 많은 여름이 | 김연수 | `alternative` |

- 파일명 stem(확장자 제외)이 ORB 매칭 후 알라딘 검색어로 쓰입니다. 띄어쓰기 없이 저장된 이름을 유지하세요.
- 표지를 바꿀 때는 같은 파일명으로 덮어쓰거나, `register.py`로 웹캠 재등록하세요.
- 표지 다운로드: `npm run refs:fetch` (알라딘 API → 이 폴더)
- 표지 확인: `npm run refs:demo`
